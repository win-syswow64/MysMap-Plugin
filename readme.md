# MysMap-Plugin

1. 全新提瓦特地图资源查询
2. 支持自动下载图片
3. 无需#更新xx在哪

# MysMap-Plugin使用

1.打开Yunzai-Bot根目录运行
Github
```
git clone https://github.com/win-syswow64/MysMap-Plugin.git ./plugins/MysMap-Plugin
```
Gitee
```
git clone https://gitee.com/catbos/MysMap-Plugin.git ./plugins/MysMap-Plugin
```

2.重启后给机器人发送`地图帮助`查看帮助
